# Ai_Academy-
 Model Evaluation & NLP Chunking Lab
 Overview

This project demonstrates key concepts in Machine Learning model evaluation and Natural Language Processing (NLP) preprocessing. It covers both classification and regression evaluation techniques, along with text chunking and embedding generation for real-world NLP applications.

 Objectives
Understand model evaluation metrics for classification and regression
Implement train-test split and cross-validation
Analyze overfitting and underfitting
Perform text chunking (word-based & sentence-based)
Generate semantic embeddings using transformer models
